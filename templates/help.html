En la página principal, describe lo que deseas probar en lenguaje natural. Incluye:

Preguntas Frecuentes

Un "Vuelo de Prueba" es una sesión de prueba automatizada donde el sistema navega por un sitio web siguiendo las instrucciones proporcionadas, como un piloto automático. Incluye:

  • Navegación automática
  • Interacción con elementos
  • Validación de resultados
  • Generación de evidencias

Actualmente soportamos las siguientes APIs de IA:

  • OpenAI (GPT-4): Recomendado para instrucciones detalladas
  • Google Gemini: Excelente para análisis visual
  • Anthropic Claude: Ideal para contextos complejos
  • Azure OpenAI: Para entornos empresariales
  • DeepSeek AI: Alternativa económica

El Modo Sigiloso (headless) ofrece varias ventajas:

  • Mayor velocidad de ejecución
  • Menor consumo de recursos
  • Ideal para pruebas en segundo plano
  • Compatible con entornos sin interfaz gráfica

Hay varias formas de acceder al código:

  • Botón "Ver Código" en resultados
  • Carpeta de scripts: /scripts/tests/
  • Historial de ejecuciones
  • Descarga directa en formato Python

Los reportes de evidencia se pueden generar:

  • En formato PDF o Word (.docx)
  • Desde la vista de resultados
  • Incluyendo todas las capturas
  • Con logs detallados
